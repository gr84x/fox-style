# Changelog

Version history and notable changes for **@gr84x/fox-style**.

For the full list of changes, see [CHANGELOG.md](https://github.com/gr84x/fox-style/blob/main/CHANGELOG.md) in the repository, or the [Releases](https://github.com/gr84x/fox-style/releases) page.

## Summary

- **0.1.0** — Initial package: design tokens, Badge, Button, ModeBanner, Storybook.
- **0.2.0** — Fundamental components: Loader (7 animations), SearchInput, SegmentedControl, Card, StatusDot, ProgressBar, NavTray, Layout. Card typography classes. SegmentedControl long-label handling. NavTray icon docs.

Future: Documentation and Foundation pages, optional components (ChatInput, PhaseDivider, SystemStatus), recipe stories.
